# Proyectia | Histórico de puntajes UNAL

Versión corregida con revisión de cupos y registros musicales instrumentales.

Cambios aplicados:
- Se incorporaron los cupos y puntajes corregidos del archivo revisado manualmente.
- Se verificaron los registros marcados con `x` en la columna de cupos.
- Los registros que no aparecían en el PDF original o tenían puntaje tomado de otro instrumento fueron eliminados.
- Cuando el programa aparece en el documento y el número de admitidos es 0 o No aplica, se asignó cupos = 0.
- Se corrigieron casos como Música Inst. - Arpa cuando el puntaje correspondía realmente a otro instrumento.
- Se mantiene la base pública sin columna `tipo`.

Control final:

- Registros regulares sin cupos: 0
- Registros PEAMA sin cupos reportados en fuente: 1386
