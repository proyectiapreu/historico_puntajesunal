# Proyectia | Histórico de puntajes UNAL

Tablero estático listo para publicar en GitHub Pages.

## Archivos principales

- `index.html`: página principal del tablero.
- `assets/proyectia_logo.png`: logo de Proyectia usado en el encabezado.
- `puntajes_unal_todas_sedes_dashboard.csv`: base consolidada de auditoría.
- `.nojekyll`: evita problemas de renderizado en GitHub Pages.

## Publicación rápida en GitHub Pages

1. Crea un repositorio en GitHub, por ejemplo: `puntajes-unal-proyectia`.
2. Sube todos los archivos de esta carpeta a la raíz del repositorio. No subas la carpeta completa como subcarpeta.
3. En GitHub, entra a `Settings`.
4. Entra a `Pages`.
5. En `Build and deployment`, selecciona:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Guarda.
7. GitHub generará un enlace público parecido a:
   `https://TU_USUARIO.github.io/puntajes-unal-proyectia/`

## Nota metodológica pública

El tablero es orientativo y no oficial. El estimado compara históricos del mismo programa, sede o PEAMA y semestre de admisión. No garantiza admisión.
